## ğŸ Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- A code editor (VS Code, Sublime Text, etc.) - optional, for viewing/editing code
- Basic understanding of HTML, CSS, and JavaScript - optional, for learning purposes

### Installation & Running

1. **Clone the repository**
   ```bash
   git clone https://github.com/Qadirdad-Kazi/internee.pk-projects-HTML-CSS-JS.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd internee.pk-projects-HTML-CSS-JS
   ```

3. **Open any project**
   - Navigate to the desired project folder
   - Open the HTML file in your web browser
   - Or use a local development server (e.g., Live Server extension in VS Code)

### Alternative: Using Live Server

If you have VS Code with the Live Server extension:

1. Open the project folder in VS Code
2. Right-click on any HTML file
3. Select "Open with Live Server"

### Project 1: Internee.pk Landing Page Clone

A comprehensive clone of the Internee.pk landing page featuring:

- **Features:**
  - Responsive navigation bar
  - Hero section with call-to-action buttons
  - Interactive carousel for internship categories
  - Multiple sections showcasing platform features
  - Task portal, LMS, and job portal information
  - Testimonials section
  - Footer with social media links

- **Technologies:**
  - HTML5, CSS3, JavaScript
  - Boxicons for icons
  - Custom animations and transitions

- **File:** `Project 1/Home.html`

### Task 3: Contact Form Page

A clean and modern contact form implementation:

- **Features:**
  - Responsive contact form layout
  - Form validation using JavaScript
  - Custom styling with CSS
  - User-friendly interface with visual feedback

- **Technologies:**
  - HTML5 forms
  - CSS3 styling
  - JavaScript validation

- **File:** `Task3/contact.html`

### Task 4: E-commerce Homepage

An e-commerce style homepage with modern design:

- **Features:**

  - Product categories dropdown
  - Image gallery/showcase

- **Technologies:**

  - Custom CSS animations
